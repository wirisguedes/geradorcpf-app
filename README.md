# Gerador de CPF
![image](https://github.com/user-attachments/assets/f08644ee-d800-4f07-a06b-de150f5495c9)
![image](https://github.com/user-attachments/assets/595cdc0f-8820-41d6-ae28-066745025400)

Este é um programa simples de **geração de CPF** desenvolvido em Java com interface gráfica utilizando o Swing. Ele permite gerar números de CPF com base em dois critérios principais: o padrão do CPF e o estado de origem.

## Funcionalidades

- **Gerar CPF por Dígito Verificador**: Permite gerar um CPF válido com base em um padrão e dígito verificador selecionados.
- **Gerar CPF por Origem**: Permite gerar um CPF com base no estado de origem e no padrão selecionado.
- **Copiar para a Área de Transferência**: Após gerar o CPF, você pode copiá-lo diretamente para a área de transferência clicando em um botão.

## Tecnologias Utilizadas

- **Java 8**: A linguagem utilizada para o desenvolvimento do programa.
- **Swing**: Framework para a construção da interface gráfica.

## Requisitos

- **Java 8** ou superior instalado.
- Sistema operacional Windows, Linux ou MacOS (com Java instalado).
